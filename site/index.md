features:
  - title: A real done-check
    details: '"Done" is never a model''s self-report. Gates pair a deterministic signal (the tests really pass) with independent judges — juries, scored dimensions, fail-closed confidence.'
  - title: The workspace is the state
    details: Every attempt starts with a fresh context. Progress lives in files and git; decisions are written into commit bodies and read back on the next attempt. Kill a run, resume it, lose nothing.
  - title: Live, steerable graphs
    details: A dag built on a live plan is data while it runs — add discovered work, cancel a doomed branch, rewire the rest. Every edit is validated before it applies and recorded after.
  - title: Preemption with dignity
    details: A graceful cancel lets the work in flight finish its turn and yield at a safepoint; the hard abort is the deadline, not the mechanism. Crystallized work is immutable.
  - title: Momentum, measured
    details: A run is alive when gated work is landing — never merely busy. The alive / idle / stalled / done read is computed from auditable events, served over HTTP, and impossible to fake.
  - title: The world steers
    details: Any webhook — an issue opened, an incident fired — is ingested, validated, filtered, and routed into the running graph as force. The same port answers back with whether it landed.
